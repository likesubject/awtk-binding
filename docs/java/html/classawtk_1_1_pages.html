详细描述

页面管理控件。

只有一个Page处于active状态，处于active状态的Page才能显示并接收事件。 常用于实现标签控件中的页面管理。

pages_t是widget的子类控件， widget_t的函数均适用于pages_t控件。

在xml中使用"pages"标签创建页面管理控件。如：

<tab_control x="0" y="0" w="100%" h="100%"
<pages x="c" y="20" w="90%" h="-60" value="1">
...
</pages>
<tab_button_group>
...
</tab_button_group>
</tab_control>

更多用法请参考： tab control
