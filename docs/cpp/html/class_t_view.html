详细描述

一个通用的容器控件。

它本身不提供布局功能，仅提供具有语义的标签，让xml更具有可读性。 子控件的布局可用layout_children属性指定。 请参考https://github.com/zlgopen/awtk/blob/master/docs/layout.md "布局参数"。

view_t是widget的子类控件，widget_t的函数均适用于view_t控件。

在xml中使用"view"标签创建view。如：

*<view x="0" y="0" w="100%" h="100%" children_layout="default(c=2,r=2,m=5,s=5)">
*</view>

可用通过style来设置控件的显示风格，如背景颜色等。如：

*<style name="default" border_color="#a0a0a0">
*<normal bg_color="#f0f0f0" />
*</style>
