详细描述

多行编辑器控件。

mledit_t是widget的子类控件，widget_t的函数均适用于mledit_t控件。

在xml中使用"mledit"标签创建多行编辑器控件。如：

*<mledit x="c" y="m" w="300" h="300" />

更多用法请参考：[mledit.xml](

https://github.com/zlgopen/awtk/blob/master/demos/assets/default/raw/ui/mledit.xml)
